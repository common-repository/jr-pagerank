== Description ==

This PageRank plugin allows you to show your Google PageRank on your blog within seconds!

You also have an option to select whether you'd like to show your homepage PageRank throughout your website or whether you'd like to show the PageRank value of each individual page!

== Installation ==

1. Upload `/jr-pagerank.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Edit the JR PageRank Settings in your Administration Panel
4. Done!
